Fundanet Multitenant

Sistema central de servicios compartidos para todo el ecosistema Fundanet

Descripción

El sistema Multitenant es el núcleo central que proporciona servicios compartidos a todas las aplicaciones del ecosistema Fundanet. Gestiona la autenticación, autorización, tenants, usuarios, licencias y configuración centralizada.

Fundanet AI Agents

Sistema de automatización inteligente para estudios clínicos

Descripción del Proyecto

Fundanet AI Agents es una solución inteligente desarrollada por Semicrol para automatizar tareas críticas y repetitivas en la gestión de estudios clínicos, reduciendo drásticamente el esfuerzo manual y los errores humanos.

El sistema transforma automáticamente la memoria del estudio clínico (PDF) en un Excel estructurado y validado para su importación directa en el sistema CTMS.

Tecnologías

Python 3.10+

FastAPI + Uvicorn

Azure AI Foundry

Plataforma IA

GPT-4o

Azure OpenAI (UE)

Fundanet AI Tickets

Sistema de gestión inteligente de tickets y gastos

Descripción

Fundanet.Tickets es una solución desarrollada por Semicrol para la digitalización y automatización del proceso de gestión de tickets y justificantes de gasto. Permite a los usuarios capturar tickets o facturas desde sus dispositivos móviles mediante una fotografía, que el sistema analiza automáticamente con IA.

Fundanet Nexus

Plataforma de interconectividad de sistemas en Centros de Investigación

Descripción

Nexus es una plataforma integral diseñada para interconectar diferentes sistemas y plataformas utilizados en los Centros de Investigación facilitando la comunicación y garantizando una integración fluida.
